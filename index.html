On-Demand Transit and Booking App

I developed a web application that utilizes machine learning models to optimize passenger scheduling for small bus vehicles and provides the transit compa book requests, cancel trips, manually edit passenger and vehicle schedules, and view overall peformance history. The application, built in Dash Plotly (react.js) and deployed on Google App Engine, features an interactive scheduling platform for fleet operators, communicates with vehicles to display real-time vehicle location and passenger activities, and provides automated tools to increase the speed and accuracy of manual trip booking, cancellation, and rerouting. The application also provides a dashboard to review and analyze vehicle or driver performance, passenger history, and integrates google authorization and secure APIs our MongoDB document database.
